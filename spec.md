purpose: 
This app is for building a community platform for students and professionals to connect and share their knowledge and experiences.

features: 
1. User authentication
2. User profile
3. Feed for posts
4. allow anonymous posts
5. allow users to like and comment on posts


mobile app flow:
1. user can login using google
2. user can create an account using google
3. user can create a post
4. user can like a post

technology stack:
1. android
2. firebase
3. firestore
4. firebase authentication
5. firebase storage


testing: 
1. unit testing
2. integration testing
3. ui testing


Development process:
Every feature should be tested with different scenarios and edge cases.
All tests should be run after changes.
Add tests for new features. Cover all the scenarios and edge cases in tests.
Verify that the app works as expected.
